Source code of all demos

Source code of all demos are in the "Tutorial" folder of the GacUI for Windows repo:

  • Hello, world! : Every possible ways to create a hello world GacUI project.
    • Cpp : Using C++ from stratch.
    • WorkflowScript : Using Workflow script from stratch.
    • CppXml : Using C++ and GacUI Xml Resource, with generated C++ code for loading UI from the precompiled binary GacUI Xml Resource.
    • Xml : Loading the precompiled binary GacUI Xml Resource dynamically at runtime.
    • MVVM : A MVVM pattern demo using Xml Resource.
  • Layout : Demonstrates all layout features.
    • Alignment : Simple layout.
    • Stack : Stack layout with configurable axis directions.
    • Flow : Flow layout with configurable axis directions.
    • Table : Table layout.
    • RichTextEmbedding : Embed controls in a rich text document.
  • Controls : Demonstrates all controls and control templates. This demo is under development.
    • ContainersAndButtons : Using container controls and button controls.
    • TextEditor : Implementing a text editor that handle plain text and XML.
  • ControlTemplate : Demonstrates all controls and control templates. This demo is under development.
    • BlackSkin : Create a set of control templates which look like Visual Studio's dark theme.
  • XmlResource : Demonstrates all features in the XML resource file. This demo is under development. (developing)
  • Scenarios : Some complete applications building by GacUI with a little more complexicity. (developing)
  • etc ...

Running a demo

Using Visual Studio 2013 or higher version to open the solution, and then run the demo by pressing F5.

If you want to modify the demo for practicing, you may notice that there are a Codegen.bat file in the UI folder in every demo projects. Modify the xml and run the bat file, the corresponding binary resource in the UIRes folder will be updated. After these steps are done, press F5 to see the result!
